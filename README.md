# Ángela Sánchez | Full Stack Portfolio

Hi there! I'm Ángela, and this is the source code for my personal portfolio. 

I developed this full-stack application with its own custom-built Content Management System (CMS) to manage my projects dynamically. The design follows a brutalist aesthetic: high contrast and straight to the point.

🚀 **Live Demo:** [https://cms-portafolio-1.onrender.com/](https://cms-portafolio-1.onrender.com/)

## What's under the hood?

**Frontend:**
- React 18 & Vite
- Tailwind CSS (Brutalist style)
- React Router & Bilingual support

**Backend & Data:**
- Node.js & Express API
- Supabase (PostgreSQL)
- JWT Authentication
- Deployed on Render
